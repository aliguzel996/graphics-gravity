## Product Description

Graphics Gravity is a creative 2D physics tool for building compositions from text, geometric shapes, PNG images and SVG artwork. Drop pieces exactly where you want them, then watch them fall, roll, collide and react inside a rotating container.

Control gravity, bounce, friction, attraction, size, color, transparency and vector outlines in real time. Export the current view as PNG, JPG or SVG, record it as WebM, save a PNG sequence, or store the complete scene as a `.graphicsGravity` project for later editing.

## Long Description

Graphics Gravity turns graphic design elements into physical objects. Add a word, a single letter, a basic shape, a transparent PNG or an SVG; click or hold anywhere inside the container to release it into the scene. Every piece has weight, momentum and collision, allowing compositions to form through falling, rolling, bouncing and pressure instead of a fixed layout.

Build the boundary from a circle, triangle, square, adjustable polygon or a custom-drawn shape. Keep it invisible, reveal its outline or rotate it automatically in either direction. Fine-tune gravity, bounciness, friction and mutual attraction while controlling global, random and per-input sizing.

The appearance system includes color and alpha controls for the preview, container, text, backgrounds, borders and vector graphics. Vector outlines have their own color, opacity and thickness, while transparent regions in imported artwork are excluded from the physical silhouette.

Choose a free, square or vertical output frame and zoom directly into the composition. Graphics Gravity exports exactly what is framed in the preview as transparent PNG, editable SVG or flattened JPG. Motion can be recorded as WebM or converted into a 30 FPS PNG-sequence ZIP. Complete scenes can be saved as `.graphicsGravity` project files and reopened in the web or Windows version.
